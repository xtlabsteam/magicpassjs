# MAGICPASS-JS

MAGICPASS-JS es un generador de contraseña con cifrado simple desarrollado en JS Vanilla, se trata
de un proyecto diseñado con el objetivo de familiarizarme con el lenguaje JS, por lo que es bastante
sencillo y no cuenta con características modernas como FrameWorks y otras características avanzadas


Su cifrado es sencillo pero útil, utiliza una combinación de 0/1 al inicio y final y entre medias le añade
símbolos especiales, además de la "magicword" y el "magicnumber", que actuan como una forma sencilla de recordar nuestras contraseñas generadas con el programa

Es un proyecto de código abierto que tiene como objetivo que vosotros también podais aprender conmigo además
de explorar libremente el código del proyecto, espero que os guste ;)
